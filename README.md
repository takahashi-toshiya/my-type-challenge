# My Type Challenges 解答記録

このリポジトリは、[type-challenges](https://github.com/type-challenges/type-challenges) のTypeScript型システムに関する問題を1日1問解いて記録するための学習用プロジェクトです。

- 目的：型の理解を深める / 日々の学習習慣を作る
- 方針：毎日1問ずつ `questions/` 以下の `template.ts` に解答を記述
- テスト：`test-cases.ts` に含まれるユニットテストで検証
- ライセンス：MIT（元のリポジトリ準拠）
- ブランチ： 毎日の解答は main ブランチに直接コミット＆Pushする
